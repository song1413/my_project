## 핵심 기능  Key Feature
- 내가 읽은 책에서 밑줄친 부분을 입력합니다.
- 밑줄친 내용을 핸드폰 시작화면에 나타냅니다.
- 내용은 24시간 간격으로 바뀝니다.
